### PESQ - P.862  
Reference implementation for ITU-T Recommendations P.862, P.862.1 and P.862.2.
Version 2.0 October 2005.  

* Mac: PESQ_MAC.bin  
* Linux: PESQ  
* Windows: pesq.exe  

PESQ score:
* Narrow Band  – Raw MOS, MOS LQO  
* Wide Band    – MOS LQO  

datatype: **int16**
